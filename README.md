
# Incident & Change Management Platform

This project demonstrates a senior-level, end-to-end system built using
Python, Java Spring Boot, React, and Docker.

It focuses on clean architecture, service separation, and controlled automation.

## Architecture

UI (React) -> API (Spring Boot) -> Engine (Python)

All services are containerized and run using Docker Compose.

## How to Run

docker compose up --build

Services:
- UI: http://localhost:3000
- API: http://localhost:8080
- Engine: http://localhost:5000
